# Liftee

## 開発言語

- Ruby3.0.1
- Rails6.0.3

<br>

## 実行手順

```plain text
$ git clone git@github.com:dai-gi/Liftee.git
$ cd Liftee
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails s
```
