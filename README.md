# Liftee

## 開発言語

- Ruby
- Ruby on Rails

<br>

## 実行手順

```plain text
$ git clone git@github.com:dai-gi/Liftee.git
$ cd Liftee
$ bundle install
$ rails db:create
$ rails db:migrate
$ rails s
```
